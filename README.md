# bookkeeping-app

## Description
Application to manage and store book records. Stores the book's title, author, year of publication, price, and whether it is available in-print or out-of-print into a SQLite database. Books may be saved, edited or deleted from the database.  

## Features
SQLite database
GUI with save, edit, and delete functionality
Object-oriented design

## Technologies Used
Microsoft Visual Studios, C# development using Winforms, SQLite Database

## How To Run
Clone the repository
Open the solution file (.sln) in Visual Studio
Build the project (Ctrl + Shift + B)
Run the program (F5)

## What I Learned
Overriding the ToString() function for an instantiated book object to be converted to a string literal for storage in the listbox
Logical connection between buttons and their functionality pertaining to the listbox
Creating a program that stores information longterm via a database
SQLite queries for table creation, modification, and entry deletion

## Future Improvements
Cleaner looking GUI
Better differentiation between the save and edit buttons, as SAVE does not save changes, it only adds a new entry.
